# Real World Examples

Datasets managed with fdl in production. All examples below are Japanese public data.

## Examples

| Dataset | Description |
|---|---|
| [dataset-calendar](https://github.com/queria-io/dataset-calendar) | Japanese calendar (holidays, weekdays, fiscal years, eras) |
| [dataset-zipcode](https://github.com/queria-io/dataset-zipcode) | Japanese nationwide postal code directory |
| [dataset-e-stat](https://github.com/queria-io/dataset-e-stat) | Japanese social and population statistics (prefecture and municipal level) |
| [dataset-reinfolib](https://github.com/queria-io/dataset-reinfolib) | Japanese real estate transaction prices |
