# Installation

Requires Python 3.13 or later.

## pip

```bash
pip install frozen-ducklake
```

## uv tool

Installs as an isolated CLI tool.

```bash
uv tool install frozen-ducklake
```

## pipx

```bash
pipx install frozen-ducklake
```

## uv add

Adds as a project dependency.

```bash
uv add frozen-ducklake
```
